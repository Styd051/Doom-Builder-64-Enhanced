- Allow Negative Tags
Doom 64 allows for negative tags, which lets macros to use negative values. So instead of storing tags as ushorts, it stores them as shorts.
The arrow incrementor and tagged highlighting will need to be updated to support negative tags eventually. by Immorpher

- Fix Make Sector Color Gradient Rounding Bug
